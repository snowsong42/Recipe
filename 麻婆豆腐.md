# 配料
1. 400g豆腐，一定不能选内酯豆腐
2. 100g牛肉馅
3. 大量：干辣椒（刀口辣椒）、花椒
4. 检查固料：大葱一根、姜、蒜、八角、盐、白砂糖、淀粉
5. 检查液料：白醋、蚝油、料酒/白酒、豆瓣酱
# 备料
- 切蒜末、切姜末，葱白切成葱花，葱叶切成小叶片。蒜末摆在一边，姜末+葱白在一起，葱叶摆在一边
- 开火把铁锅烧热，小火甚至关火，烘焙花椒粒，至香气四溢，放到一个碗里
- 干辣椒剪段，放在碗里，开火把铁锅烧热，小火甚至关火，尽可能少放辣椒籽，把辣椒段放进锅中烘焙，辣椒呈橘红色。和花椒粒放到一个碗里，碾碎。
- 勾兑一碗淀粉水
- 给牛肉馅再剁几刀
# 焯水
豆腐切块。老豆腐冷水下锅，嫩豆腐热水下锅，加一勺盐，煮至沸腾，捞出豆腐
>这一步可以先做，等烧开水的同时去备料
# 炸香
- 加**大量**的菜籽油，几乎沫过牛肉馅为好。牛肉馅炸至酥脆，捞出。
- 小火，先加蒜末，煸香，至微微焦黄，加入葱白和姜末。翻炒几下，加入豆瓣酱约三勺，疯狂翻炒，
# 炖菜
- 炒香后加清水，豆腐下锅。加一勺半味精，半勺白糖，一勺耗油，将花椒-辣椒末放一半进去。
- 大火炖一会儿，汁水稍稍收紧，准备勾芡：分批少量加淀粉水，每次加都勤搅拌。
- 另开一个灶（其实盛出来用这个锅也行），烧油，将热油倒在剩下的半碗花椒-辣椒末里，制备热辣椒油。
- 炸好的肉末，葱叶都放进豆腐里，淋热辣椒油，出锅。
