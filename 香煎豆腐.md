# 配料
1. 豆腐
2. 一个鸡蛋
3. 检查固料：香葱、小米辣、蒜、盐、白砂糖、淀粉
4. 检查液料：蚝油、酱油
# 备料
豆腐切片，越薄越好，但也不能太薄吧反正。放在盘子里，倒入一个搅拌碎的鸡蛋，抹匀，加淀粉，抹匀。
切小香葱末，切小米辣末，切蒜米
调料汁：
- 蒜末、小米辣、香葱
- 蚝油1勺
- 酱油2勺
- 糖1茶匙
- 盐1茶匙
- 半碗清水
# 香煎
煎豆腐至两面金黄
# 炖煮
倒入料汁，水烧开，略微收汁即可。
