提到的技巧
# 原料
1. 一斤五花肉
2. 酌情添加：鸡蛋、土豆、青椒
3. 检查固料：葱白、姜、蒜、香叶、八角、桂皮、白砂糖、冰糖
4. 检查液料：生抽、老抽、白醋、蚝油、料酒/白酒
# 备料
烧一锅开水，五花肉整块下锅，加白醋煮一会儿，捞出。
切肉块差不多指头大小，大拇指或食指大小任意。
切葱白约一掌长，对半切开，一半用于焯水，一半用于炖肉。将切得的两段都剖开，再来一刀
切姜片7片，4小片用于焯水，3大片用于炖肉。
剥蒜4瓣左右。
**提前拌好调料：** 生抽2勺、老抽1勺、耗油1勺、醋1勺、白砂糖半勺、料酒1勺
# 焯水
冷水下肉块，加入葱姜，加入适量料酒，中火煮5分钟。
（若要吃鸡蛋，同时煮起来鸡蛋，煮大约9分钟，也就是在炒制的时候择机关火）
撇血沫，捞出肉块，注意尽量沥干水分。
# 炒制
*抓住开火前的时机，就在这时开始烧一壶开水！！！*
锅中下油，热油，中等油温下五花肉。
>虽然五花肉出油，未来油会过剩乃至需倒掉，但第一步的油不能省。
>必须先有一定的油，五花肉在油里煎，才能出油！这或许类似于"点火条件"，一定要在初始油干涸前煎出油，否则就会缺油粘锅。

炒至略金黄，肉块出锅。
炒糖色：倒掉一些油，关火，等待油温下降。不开火，抓一把冰糖下锅，稍微炒一炒。开最小火，不断搅拌翻炒，捣碎大块儿冰糖。注意时不时把锅从火上掂起来降温，顺便晃一晃锅。炒至枣红色，糖熔化为与油分层并存的糖浆，下入五花肉翻炒，使肉块均匀裹上红褐色糖浆。
开中火，下葱、姜、4瓣左右蒜、3片左右香叶、2块左右八角、一个指节长的桂皮，倒入一整碗提前拌好的料汁，稍微炒一炒
# 炖肉
(若有鸡蛋，此时可以剥鸡蛋。冷水水流冲洗更好剥，剥好放进高压锅)。
将肉倒入高压锅，或普通炖肉锅。(若有土豆，此时可以下土豆)。加水淹没肉，小火慢炖。
- 高压锅20分钟
- 普通锅40分钟
临出锅，开大火，快速晃动锅中的肉，防止糊底，收汁差不多到肉的一半，不再会完全淹没肉即可。可以挑出一些葱姜蒜八角桂皮等固料，防止食客误食。出锅！
