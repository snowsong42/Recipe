# 原料
1. 一斤鸡肉
2. 青椒约4根、青红椒约2根
3. 土豆两块
4. 检查：姜、蒜、白芷、盐、胡椒粉、白糖、味精、香油、料酒/白酒
5. 酱料：黄豆酱+甜面酱，或者其他组合
# 腌制
用一勺盐+胡椒粉腌制鸡肉
剥蒜约大半头，切姜片和辣椒片（留两根青椒），切土豆块
# 煸鸡肉
先煸姜片至金黄，下入鸡肉.
水明显减少，鸡肉微微焦黄——下蒜
开始粘锅——下白芷，约三四片。稍微煎一会儿，加料酒。下酱，**开小火**翻炒
> 鸡肉要大火煸干煸香，酱必须小火否则极其容易糊

耗油，加味精两勺，白糖一勺，老抽调色，加入两根青椒，翻炒翻炒
# 煲鸡肉

加水没过鸡肉，如果要加土豆，需更多水，并在此时下入土豆。
大火收干汁水，挑出煮的整根辣椒，加入辣椒片。再次翻炒。撒上少量香油，出锅。
