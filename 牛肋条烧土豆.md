# 原料
- 一斤牛肋条
- 小土豆，或者切成小块的大土豆。
3. 检查固料：姜、蒜、白芷、八角、桂皮、大量干辣椒、小米辣、辣椒粉、冰糖、盐、白胡椒粉、味精、小香葱。
4. 检查液料：生抽、老抽、蚝油、料酒/白酒、豆瓣酱
# 备料
小土豆不用切开，大土豆一定要切碎！这是一个教训。
切姜片，约6片。大蒜约10瓣，4瓣直接用，6瓣做蒜末。小米辣切碎。
准备一碗香料：大量干辣椒、白芷、八角、桂皮
# 焯水
冷水下锅。给一点料酒，大火煮开之后开小火，煮到不出浮沫为止，大约要15min~20min。盛出来之后用热水清洗。
# 煸炒
加油，姜片下锅，煸香。下牛肋条，下整颗蒜，煸至焦褐，把香料一股脑倒进去。加盐入底味。一斤半牛肋条加上一斤土豆是两大勺盐。
香料煸香之后加料酒，转入小火，加辣椒粉，豆瓣酱，炒酱炒香之后加老抽（3g），蚝油（4g），炒匀之后开大火，下入土豆和清水。
加入冰糖、白胡椒粉、味精。然后普通锅炖1小时，高压锅炖45分钟。盛出。
# 重口
起新锅烧油，红油花椒油可选，炒蒜蓉和小米辣，加辣椒粉，制成蒜蓉辣椒油。把炖好的肉倒进来，大火收汁。尝尝味道，适当用生抽调味。
出锅，撒上一把葱花。
